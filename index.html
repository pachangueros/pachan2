Un solo partido:

        Ganador  -> baja -1
        Perdedor -> sube +1

        Segundo -> baja -1
        Tercero -> baja -0,5
        .....       
        Antepenúltimo -> sube +0,5
        Penúltimo -----> sube +1
        Último --------> sube +1,5     1222222
